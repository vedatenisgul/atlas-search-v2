Zero-dependency search infrastructure

A self-contained crawler and real-time search engine built on Python's standard library.

Atlas Search replaces Scrapy, Elasticsearch, Redis, and PostgreSQL with hand-rolled, thread-safe, in-memory primitives: an RLock-guarded trie, a SHA-256 deduped frontier, and a JSON-flushed NoSQL store.
